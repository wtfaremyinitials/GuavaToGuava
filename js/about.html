About GuavasToGuavas

This game is a clone of Apples to Apples by Mattel coded in Java. The backend of this game is utilizes two libraries: JSON and HttpServer, along with Java of course. The frontend is coded in HTML, CSS, and JavaScript.

How did arrive at this idea?

Apples to Apples is a great game, but it was missing one thing: a digital interface. So we set out to change this. With GuavasToGuavas, you cant play with your friends seamlessly.
